## About Me

I'm a Frontend Developer based in **Lahore, Pakistan** with 4+ years of experience turning ideas into production-ready web applications. I specialise in React.js and Next.js, but my work often goes beyond the UI — from wiring up Shopify storefronts and writing SEO metadata, to building full Zoho One automation pipelines for real businesses.

I care about writing code that's readable, maintainable, and actually solves the problem — not just code that works.
